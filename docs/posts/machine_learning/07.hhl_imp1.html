HHL Algorithm Implementation in Qiskit

Introduction

This document provides a step-by-step implementation of the HHL algorithm using Qiskit. The algorithm is broken into four key phases:

  1. *State Preparation
  2. Quantum Phase Estimation (QPE)
  3. Controlled Rotation & Measurement of the Ancilla Qubit
  4. Uncomputation (Inverse QPE)

Phase 0: Environment Setup

Overall

\(A\vec{x}=\vec{b}~~\rightarrow~~\vec{x}=A^{-1}\vec{b}\)
The goal is to get \(\vec{x}\)

\(|x\rangle = A^{-1}|b\rangle=\sum\limits^{N-1}_{j=0}b_jA^{-1}|u_j\rangle=\sum\limits^{N-1}_{j=0}\frac{b_j}{\lambda_j}|u_j\rangle\) \[ \begin{aligned} &~~~~~~A|u_j\rangle = \lambda_j|u_j\rangle \\ &\rightarrow A^{-1}A|u_j\rangle = A^{-1}\lambda_j|u_j\rangle \\ &\rightarrow |u_j\rangle = A^{-1}\lambda_j|u_j\rangle \\ &\rightarrow A^{-1}|u_j\rangle = \frac{1}{\lambda_j}|u_j\rangle \end{aligned} \]

Variables

\(n_b\): # of qubits in b-register

\(N_b=2^{n_b}\): # of unknows (length of \(\vec{b}\), \(len(\vec{b})=len(\vec{x})\))

\(n_c\): # of qubits in c-register

a larger \(n_c\) results in higher accuracy when the encoding is not exact

\(N=2^n\)

Propreties

HHL is \(O(\log(N))\)

Phase 1: State Preparation

Initial State

Given \(n_b\), \(n_c\), there are total \(n_b+n_c+1\) qubits \[ \begin{align} |\psi_0\rangle &= |0\dots0\rangle_b|0\dots0\rangle_c|0\rangle_a \\ &= |0\rangle^{\otimes n_b}|0\rangle^{\otimes n}|0\rangle \end{align} \]

Prepare \(|b\rangle\)

In the state preparation, \(|0\dots 0\rangle_b\) in the b-register needs to be rotated to have the amplitudes correspond to the coefficients of \(\vec{b}\). \(\vec{b}=\left(\begin{matrix}\beta_0\\\beta_1\\\vdots\\\beta_{N_b-1}\end{matrix}\right)\Leftrightarrow\beta_0|0\rangle+\beta_1|1\rangle+\dots+\beta_{N_b-1}|N_b-1\rangle=|b\rangle\) \(|\psi_1\rangle=|b\rangle_b|0\dots0\rangle_c|0\rangle_a\)

def state_prep(mat_A, vec_b, n_c=1):
  N_b = vec_b.shape[0]
  n_b = np.round(np.log2(N_b))
  qr_b = QuantumRegister(n_b, name='qr_b')  # b-register
  qr_c = QuantumRegister(n_c, name='qr_c')  # c-register
  qr_a = QuantumRegister(1, name='qr_a')    # a-register
  qc = QuantumCircuit(qr_c, qr_b, qr_a)  # complete circuit

  norm_b = vec_b / np.linalg.norm(vec_b)
  qc.initialize(norm_b, qr_b)
  return qc, qr_b, qr_c, qr_a

mat_A = np.array([[1, -1/3], [-1/3, 1]])
vec_b = np.array([0,1])
